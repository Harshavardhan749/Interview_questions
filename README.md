1. Data Structures (The Basics)

Array Methods: .map(), .filter(), .reduce(), .sort(), and .splice().
String Methods: .split(), .join(), .slice(), and regex basics.
Math: Math.max(), Math.floor(), and handling randomness.

3. Basic Algorithms
Standard patterns every beginner should know.
Searching: Linear search vs. Binary search.
Sorting: Introduction to Bubble Sort and Insertion Sort.
Two Pointers: A common technique for array-based interview questions.
